# Ideaflow Commands (Addon)

This package is an **addon** for the Ideaflow skill. It provides slash commands for listing and browsing ideas.

> Requires the **Ideaflow skill** to be installed first.

## Installation

```
openclaw plugins install @benmillerat/ideaflow-commands
```

## What it adds

- `/ideas` — list the 6 most recent ideas
- `/ideas_status` — browse ideas by stage with inline buttons
